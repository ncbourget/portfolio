FaceBook Marketplace Gallery

Web • 3D • Animation • Game

Winter 2026

01 Installation 02 Website 03 Animation 04 Installation visitors

FaceBook Marketplace Gallery is a web-based 3D gallery and game environment built from the strange visual economy of online resale images.

The project treats marketplace photography as exhibition material, using animation, navigation, and spatial staging to turn found objects into a playable gallery system.
